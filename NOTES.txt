The Bootcamps class will produce instances of NYC bootcamps with the following attributes:
name
url
about
courses
cost ranking
hiring %


The name and url attribute will be scraped from the main page: https://www.switchup.org/locations/nyc-coding-bootcamp and assigned to each
bootcamp instance during initialization. All instancess will be added to @@all array.

Then the array will be iterated over and each instance will be assigned other 4 attributes scraped from respective urls.
